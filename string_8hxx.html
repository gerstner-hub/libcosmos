Split a string at separator boundaries.

The input string str will be split up at every occurence of the sep substring. The resulting parts are returned as a vector. By default subsequent occurences of sep are ignored. If SplitOpts::KEEP_EMPTY is set in flags then empty elements will be returned for these occurences instead.
